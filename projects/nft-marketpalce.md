# nft-marketpalce

## 基础概念
+ ask 卖方愿意出售证券的价格  
+ bid 是买方愿意购买证券的价格  
在交易市场中，ask价格通常高于bid价格

## 1.1. start
```bash
mkdir nftmarketpalce
cd nftmarketpalce
npx create-next-app .

npm install react-icons

```
